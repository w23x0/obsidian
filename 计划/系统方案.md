## 需求 2：笔记爱好模板

做一个 `笔记偏好.md` 文件，定义你喜欢的笔记风格。

### 示例结构
```
# 笔记偏好.md

## 风格
- 简洁、逻辑清晰
- 多用列表、小标题
- 不要废话

## 笔记类型模板
- 学习笔记：概念 + 例子 + 常见坑
- 项目笔记：目标 + 进度 + TODO
- 日志：日期 + 关键事件 + 感想

## 偏好
- 代码块用 ```
- 重点用 **加粗**
- 不要太多 emoji
```

---

## 需求 3：学习日志记录

用 `memory/` 目录下的文件记录，我会定期整理到 MEMORY.md。

### 流程
1. 你反馈：每次学完，告诉我「今天学了什么、进度如何、有什么问题」
2. 我记录：写到 `memory/YYYY-MM-DD.md`
3. 定期整理：把重要的进度、经验总结到 MEMORY.md

### 示例日志条目
```
## 2026-02-25
- C++：Cherno 视频看到第 115 集，学了智能指针
- 线性代数：MIT 18.06 看完第 5 讲，理解了矩阵乘法
- 问题：智能指针的 move 语义还不太熟
```
