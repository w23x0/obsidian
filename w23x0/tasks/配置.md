## 常用命令

### 读取任务
```bash
# 列出所有任务列表
gog tasks lists list

# 列出指定任务列表的任务
gog tasks list &lt;tasklistId&gt;

# 列出"我的任务"
gog tasks list MTE4NzczMzI0OTkyODAzMjQ0MDQ6MDow
```

### 添加任务
```bash
# 添加任务到"我的任务"
gog tasks add MTE4NzczMzI0OTkyODAzMjQ0MDQ6MDow --title "任务标题" --notes "任务描述"
```
