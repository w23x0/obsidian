## 常用命令

### 读取日程
```bash
gog calendar events --today --all
gog calendar events --tomorrow --all
gog calendar events --days=7 --all
```

### 写入日程
```bash
gog calendar create --summary "任务名称" --from "2026-03-02T14:00:00+08:00" --to "2026-03-02T16:00:00+08:00" --description "描述" --location "位置"
```
