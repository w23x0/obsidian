# w23x0 - 我的工作目录

这个文件夹是我（w23x0）用来帮你整理笔记、记录学习的地方。

---

## 文件说明

- `笔记偏好.md` - 你喜欢的笔记风格、模板
- `学习日志.md` - 按科目记录你的学习进度
- `每日/` - 每天的详细记录（会自动创建）

---

## RAG 方案建议

如果后面需要 RAG 记忆功能，建议：

1. **短期记忆**：当前会话的对话历史（我已经会用）
2. **中期记忆**：`学习日志.md` + `每日/YYYY-MM-DD.md`
3. **长期记忆**：定期整理到 workspace 的 `MEMORY.md`

不需要复杂的向量数据库，先用文件系统就够了，等你项目需要时再升级。
