### 正定矩阵的判断
前提必须对称矩阵

矩阵$A=\begin{bmatrix}a&b\\b&d\end{bmatrix}$，判断其正定性方法

1. 矩阵的所有特征值大于零：$\lambda_1>0,\ \lambda_2>0$；
2. 矩阵的所有顺序主子阵（左上角）的行列式大于零：$a>0,\ ac-b^2>0$；
3. 矩阵消元后主元均大于零：$a>0,\ \frac{ac-b^2}{a}>0$；
4. $x^TAx>0$ （二次型测试）（核心出装）
5. $S=A^TA$ A列独立 （最小二乘相关）

### 正定矩阵
主元-行列式-特征值都是正的
矩阵为$A=\begin{bmatrix}2&6\\6&20\end{bmatrix}$ $\det A=4$，$tr(A)=22$，特征向量均大于零
函数$f(x_1,x_2)=2x_1^2+12x_1x_2+20x_2^2$
对于$z=2x^2+12xy+20y^2$
$(0,0)$点函数的一阶偏导数均为零 二阶偏导数均为正 函数在改点取极小值
在微积分中，一元函数取极小值需要一阶导数为零且二阶导数为正$\frac{\mathrm{d}u}{\mathrm{d}x}=0, \frac{\mathrm{d}^2u}{\mathrm{d}x^2}>0$
线性代数中多元函数$f(x_1,x_2,\cdots,x_n)$，取极小值需要二阶偏导数矩阵为正定矩阵
#### 配方与消元
我们对函数配方
$f(x,y)=2x^2+12xy+20y^2=2\left(x+3y\right)^2+2y^2$
矩阵的消元，$\begin{bmatrix}2&6\\6&20\end{bmatrix}=\begin{bmatrix}1&0\\-3&1\end{bmatrix}\begin{bmatrix}2&6\\0&2\end{bmatrix}$，这就是$A=LU$，
矩阵$L$中的项与配平方中未知数的系数有关
主元与平方项外的系数有关
这就是正数主元得到正定矩阵，即**配方和消元关联**

### 正定矩阵应用

#### 能量 $x^TAx$与万能公式 $S=A^TA$
现实我们通过公式$S=A^TA$来创造对称矩阵
然后用$x^TAx>0$判断正定性
$x^TSx=(Ax)^T(Ax)=||Ax||^2>0$ 
$x\neq0$ 时严格 >0 才严格正定
要Ax=0只有0解则零空间为0，A的列向量线性无关

最小二乘法[[16投影矩阵和最小二乘法]]$A^TA\hat{x}=A^Tb$就是正定矩阵

#### $S=LDL^T$ 与 Cholesky（乔列斯基）分解
计算机是如何利用正定性极速求解的

1. 回到消元法：对于任何对称矩阵 $S$，我们在消元时可以把它分解为 $S = L D L^T$。其中 $L$ 是下三角的消元系数矩阵（对角线全是 1），$D$ 是存放**主元（Pivots）** **的对角矩阵** 
2. 正定性带来的奇迹：正定矩阵有一个极为严苛的条件——所有主元都必须严格大于 0  正是因为它们是正数，在数学上我们可以安全地对 $D$ 中的每一个元素开平方根，得到对角矩阵 $\sqrt{D}$ 
3. 乔利斯基分解（Cholesky Factorization）：把 $D$ 拆开后，原来的分解变成了 $S = L (\sqrt{D}\, \sqrt{D}) L^T$ 。利用结合律，我们令一个新矩阵 $C = L \sqrt{D}$（因为 $L$ 是下三角，乘以对角阵后依然是下三角）。相应的，右边的 $\sqrt{D} L^T$ 恰好就是 $C^T$。于是，$S$ 被完美地拆解成了 $S = C C^T$！（注：在 MATLAB 算法中，通常习惯提取上三角矩阵，所以写成 $S = C^T C$，但原理完全一致   
4. MATLAB 中的神级命令：你提到的 $A = \text{chol}(A)$（更准确地说在书里是 $C = \text{chol}(S)$），就是调用这个算法。由于正定矩阵的优良性质，计算机在做正定矩阵的消元时，绝对不需要做任何“行交换（Row exchanges）”，主元永远不可能为零，计算过程极度稳定 这使得 Cholesky 分解成为计算机求解大规模对称方程组时最快、最节省内存的方法。  


两个知识点在逻辑上形成了一个闭环：  
- 列独立的矩阵 $A$，构造正定矩阵 $S = A^T A$  
- 正定矩阵 $S$，开平方（Cholesky 分解），拆解回一个矩阵乘以它的转置 $S = C C^T$ 的形式
