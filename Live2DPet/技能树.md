# 技能树.md - Live2D 桌面宠物项目（C++ 路线）

按领域分，按学习顺序排。

---

## 领域 1：C++ 基础（先扎实这个）

### 当前在学：Cherno C++ 视频
- 完成剩余课程
- 重点掌握：
  - 指针、引用、内存管理
  - STL 容器（vector、map、string 等）
  - 类与对象、继承、多态
  - 现代 C++（智能指针、lambda、移动语义）

### 补充
- **CMake** - 构建系统，必须会
- **Git** - 版本控制

---

## 领域 2：桌面窗口与图形 API

### 方案 A：Qt（推荐，跨平台，文档全）
- **Qt 基础** - QWidget、QMainWindow
- **无边框/透明窗口** - Qt::FramelessWindowHint、WA_TranslucentBackground
- **Qt 事件系统** - 鼠标事件、绘图事件

### 方案 B：Win32 API（纯 Windows，原生）
- **Win32 窗口创建** - WNDCLASSEX、CreateWindowEx
- **无边框/透明窗口** - WS_POPUP、Layered Window
- **消息循环** - GetMessage、DispatchMessage

### 图形渲染（二选一）
- **OpenGL** - 跨平台，资料多
- **DirectX** - Windows 原生，性能好

---

## 领域 3：Live2D Cubism SDK for Native

### 基础
- **Live2D 概念** - 模型、动作、表情、参数
- **Cubism SDK 文档** - https://docs.live2d.com/cubism-sdk-manual/
- **SDK 下载与配置** - 把 SDK 集成到你的 CMake 项目

### 核心技能
- **模型加载** - 读取 .moc3 模型文件
- **渲染循环** - 每一帧更新参数、绘制
- **动作/表情控制** - 播放 motion、控制表情参数
- **鼠标交互** - 点击检测、拖拽

---

## 领域 4：pjsk 模型处理

- **模型格式** - Live2D Cubism 3.0+（.moc3）
- **模型下载** - 从 sekai.best 获取模型
- **模型管理** - 本地库、列表、预览、切换

---

## 领域 5：AI 接入

### LLM 对话
- **HTTP 客户端** - libcurl、cpp-httplib 或 Qt Network
- **JSON 解析** - nlohmann/json
- **对话上下文管理** - 历史记录、token 控制

### TTS 语音
- **TTS API 调用** - 同样用 HTTP 客户端
- **音频播放** - Qt Multimedia、SDL_mixer、或 WASAPI（原生）

### RAG 记忆（进阶）
- **向量数据库** - SQLite + faiss，或 chromadb（通过 HTTP 调用）
- **文本嵌入** - 调用 Embedding API
- **检索 + 生成**

---

## 领域 6：交互系统

- **鼠标监控** - 移动、点击、拖拽（Qt 或 Win32 事件）
- **情绪系统** - 简单的情绪标签 → 动作/表情映射
- **触发逻辑** - 点击触发、定时触发、对话触发

---

## 领域 7：OpenClaw 接入

- **OpenClaw 插件开发** - 看 OpenClaw 文档
- **进程间通信 (IPC)** - 命名管道、Socket、或共享内存
- 或者通过 **HTTP API** 和 OpenClaw 交互

---

## 学习顺序（C++ 路线）

1. **继续学完 C++ 基础**（Cherno 视频），同时学 **CMake**
2. **学 Qt**（或者 Win32），先跑通一个带窗口的 Hello World
3. **学 OpenGL/DirectX 基础**（或者直接用 Qt 的 QPainter 先凑合用，后期再换）
4. **集成 Live2D Cubism SDK**：先能加载并显示一个模型
5. **加鼠标交互**：拖拽、点击
6. **接 LLM**：用 libcurl + nlohmann/json 调用 API
7. **接 TTS**：能播放语音
8. **做 pjsk 模型管理**：下载、预览、切换
9. **加情绪系统**
10. **接 OpenClaw**（可选）
